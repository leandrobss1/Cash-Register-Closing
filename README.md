# Projeto sobre Sistema de Fechamento de Caixa

O projeto consiste em um sistema web/mobile para me auxiliar no fechamento de caixa diário.

A ideia veio através de uma solução para um fechamento mais acelerado, pois ao contar as moedas e notas do caixa você acaba tendo que contar e já multiplicar ou somar, mas com um sistema onde você apenas insere a quantidade e ele automaticamente calcula para você é bem mais rapido no dia a dia.
